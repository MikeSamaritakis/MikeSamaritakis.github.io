# mikesamaritakis.github.io

Personal website of **Michail Samaritakis** – a Computer Science undergraduate focused on conversational & generative AI, data engineering, and VR UX insights.

The site is built as a simple, fast, static portfolio and is hosted via **GitHub Pages** at:

> https://mikesamaritakis.github.io

---

## Tech stack

- **HTML5** – static pages (`index.html`, `about.html`)
- **CSS3** – custom styling (no framework), in `assets/css/main.css`
- **JavaScript** – a tiny script for footer year (and future enhancements) in `assets/js/main.js`
- **GitHub Pages** – hosting

No build step, no bundler. Just static files.
